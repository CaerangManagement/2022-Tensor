# 1. 두 자연수 a, b를 입력받아 a / b 결과를 소수점 20째자리까지 출력하는 프로그램을 작성해보세요. 

### 입력 형식 
첫 번째 줄에 a, b가 공백을 사이에 두고 주어집니다.
- 1 ≤ a, b ≤ 100

### 출력 형식
- a / b 결과를 소수점 20째자리까지 출력합니다.

### 예제1

```:python
3 5
```

```:python
0.60000000000000000000
```

### 예제2
```:python
3 7
```

```:python
0.42857142857142857142
```
